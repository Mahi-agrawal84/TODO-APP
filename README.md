# 📝 Todo App

A simple and efficient Todo Application built to manage daily tasks. This app allows users to add, update, delete, and track tasks easily with a clean and user-friendly interface.

---

## 🚀 Features

- ✅ Add new tasks
- 📝 Edit existing tasks
- ❌ Delete tasks
- ✔️ Mark tasks as completed
- 📋 View all tasks in one place
- 💾 Persistent storage (if using database/local storage)

---

## 🛠️ Tech Stack

**Frontend:**
- HTML
- CSS
- JavaScript

**Backend (if used):**
- Node.js
- Express.js

**Database (if used):**
- MongoDB / Local Storage

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/your-username/todo-app.git
cd todo-app

2️⃣ Install dependencies
npm install

3️⃣ Setup environment variables
Create a .env file in root directory and add:

PORT=5000
MONGO_URI=your_mongodb_connection_string

4️⃣ Run the application
npm start

OR (for development)
npm run dev

🌐 Usage
Open browser and go to:
http://localhost:5000

Add your tasks
Manage them easily
